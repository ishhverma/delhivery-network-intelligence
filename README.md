# Delhivery Network Intelligence 🚚📊
Graph-based logistics optimization project analyzing 144K+ trips to improve ETA accuracy and reduce bottlenecks.

## 🔍 Key Features
- Graph-based hub bottleneck detection
- ETA error analysis (MAE improvement potential)
- FTL vs Carting decision engine
- Operational cost savings estimation

## 📊 Impact
- ₹56 lakh/year potential savings
- 47% reduction in late deliveries
- ETA MAE reduction roadmap: 6.13 → <4.5 hrs

## 🧠 Techniques Used
- Graph theory (betweenness centrality)
- Operational analytics
- Route optimization
- Predictive modeling insights

## 📁 Structure
- report/ → Detailed business memo
- notebooks/ → Python analysis
- images/ → Visualizations
